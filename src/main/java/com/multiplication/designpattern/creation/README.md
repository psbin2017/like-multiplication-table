# 생성 패턴

> 객체를 생성 또는 합성하는 방법이나 객체의 표현 방법과 시스템을 분리한다.

생성 패턴을 이용하면

1. *무엇*이 생성되고,
2. *누가* 이것을 생성하며,
3. 이것이 *어떻게* 생성되는지,
4. *언제* 생성할 것인지

에 대한 유연성을 확보할 수 있다.

## 추상 팩토리 (Abstract Factory)
