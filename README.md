## 구구단 방법

알고리즘의 경우, 코드 다 지우고 시작. 가능하면 5분 이내로 시간 제한

1. 동일한 맥락을 가진 코드면 7단 ~
2. 각 `README.md` 헤더 보고 구현시 5단 ~
3. 각 `README.md` 케이스 보고 구현시 3단 ~
4. 구현 못하면 1단 및 횟수 초기화

## 구구단 일지

- 구구단 지수: 구구단처럼 자연스러운 지수, 1단이 가장 익숙하지 않고 9단은 바로 튀어나올수 있는 수준.
